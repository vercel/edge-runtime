## Install

Using npm:

```sh
npm install @edge-runtime/user-agent --save
```

or using yarn:

```sh
yarn add @edge-runtime/user-agent --dev
```

or using pnpm:

```sh
pnpm install @edge-runtime/user-agent --save
```
