## Install

Using npm:

```sh
npm install @edge-runtime/jest-environment --save
```

or using yarn:

```sh
yarn add @edge-runtime/jest-environment --dev
```

or using pnpm:

```sh
pnpm install @edge-runtime/jest-environment --save
```
