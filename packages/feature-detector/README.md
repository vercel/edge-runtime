## Install

Using npm:

```sh
npm install @edge-runtime/feature-detector --save
```

or using yarn:

```sh
yarn add @edge-runtime/feature-detector --dev
```

or using pnpm:

```sh
pnpm install @edge-runtime/feature-detector --save
```
